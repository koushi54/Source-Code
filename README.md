# Source-Code

これまでに制作した Unity / C# プロジェクトのプログラムを、ポートフォリオとしてまとめたリポジトリです。

ゲーム本体のプロジェクト全体ではなく、担当・制作したスクリプトを中心に掲載しています。UI 制御、状態管理、演出、デバッグ支援、ScriptableObject によるデータ管理など、ゲーム開発で実装した処理を確認できます。

## 掲載作品

| 作品名 | 主な掲載内容 |
| --- | --- |
| `Erase Itself` | ポーズ処理、ゲームクリア / ゲームオーバー演出、インゲーム UI、ボタンアニメーション |
| `Relay Shift` | タイトル画面、設定パネル、状態遷移、リザルト処理、画面演出 |
| `リグカジノ` | ロビーのデバッグ支援、カジノ画面、クイズ処理、リザルト演出、ScriptableObject によるデータ管理 |

## 使用技術

- Unity
- C#
- Visual Studio Code
- DOTween / LitMotion などのアニメーションライブラリ
- UniTask による非同期処理
- Netcode for GameObjects を利用したオンライン通信
- ScriptableObject によるゲームデータ管理

## 実装例

### UI / 演出

- ゲームクリア、ゲームオーバー、リザルト画面などの演出制御
- ボタンホバーやパネル表示などの UI アニメーション
- TextMeshPro を使ったテキスト演出

### 状態管理

- 設定パネルの開閉、項目選択、スライダー調整、終了確認を State パターンで管理
- シーン遷移後の状態更新や、ゲーム進行に応じた画面切り替え

### データ管理

- `ScriptableObject` を使ったゲームデータ、クイズデータ、所持金、制限時間などの管理
- インスペクターから調整しやすい形でのパラメータ設計

### デバッグ支援

- デバッグ用 UI の表示
- プレイヤーごとの所持金変更
- 特定シーンへの遷移支援
- オンライン環境での動作確認を補助する処理

## 注意

このリポジトリには、Unity プロジェクト全体ではなくスクリプトを中心に掲載しています。そのため、単体で Unity 上で完全に動作することを目的とした構成ではありません。
